가이드

생성기·프리셋·쇼케이스를 효과적으로 쓰는 방법과, 글래스모피즘을 실서비스에 적용할 때 알아두면 좋은 팁을 정리했습니다.

1. 생성기로 값 맞추기

슬라이더로 블러·투명도·테두리·반경·그림자·틴트 색을 조절하면 미리보기 카드와 하단 생성된 CSS가 함께 갱신됩니다. 최소 테두리는 유리 느낌을 위해 1px solid rgba(255,255,255,0.2) 기준으로 유지됩니다.

  • 배경이 단조로우면 블러가 잘 안 보입니다. 이미지 URL을 넣거나 쇼케이스처럼 색 덩어리를 두세요.
  • backdrop-filter는 카드 뒤에 콘텐츠가 있을 때 효과가 큽니다.

2. 프리셋 → 생성기

프리셋 메뉴에서 카드를 클릭하면 해당 스타일이 슬라이더와 미리보기에 바로 반영되고, 자동으로 생성기 탭으로 이동합니다. 이후 세부 값만 조정해 나가면 됩니다.

3. 복사한 CSS 붙여넣기

생성기 하단의 복사 버튼으로 클립보드에 스니펫을 담은 뒤, 원하는 클래스 이름을 프로젝트에 맞게 바꿔 사용하세요. ::before 하이라이트는 유리 광택용입니다. 

<div class="glass-card">콘텐츠</div>

4. 브라우저와 접근성

Safari·Chrome 등 최신 브라우저는 backdrop-filter를 잘 지원합니다. 구형 브라우저를 고려하면 불투명 배경색을 fallback으로 두는 것이 안전합니다. 대비가 낮은 글래스 위의 텍스트는 WCAG 기준에 맞게 색을 조정하세요.

5. 다크 / 라이트 모드

헤더 오른쪽의 달·해 아이콘으로 테마를 전환할 수 있습니다. 선택은 localStorage에 저장되어 다음 방문 시에도 유지됩니다.
